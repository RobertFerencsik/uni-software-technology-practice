# Bult-In Types

Dynamically typed language: not require explicit type declaration

Types:

int

float

bool

str

list

tuple

dict

set

Type hints: optional do not affect runtime behaviour
Form: variable_name: type_hint = value

Multiple Assignement: a,b = 1,2

Swapping values: a,b = b,a

Unpacking: x,y,z = [1,2,3]

Ignoring values: x,_,z = [1,2,3]

Constant: CONSTANT_NAME = value

[Operators](https://docs.python.org/3/library/operator.html): There is a module, also dunder methods, and the intrinsic operation. < >
